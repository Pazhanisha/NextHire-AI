# 🚀 NextHire AI

> AI-powered career platform that helps students prepare for jobs with resume analysis, AI interviews, and personalized career guidance.

## ✨ Features

### 📄 AI Resume Analyzer
- Upload resume PDF
- AI analyzes resume quality
- Gives improvement suggestions
- Resume history tracking

### 🎤 AI Interview Simulator
- Voice based interview practice
- AI generated questions
- Performance feedback
- Interview history

### 🧠 AI Career Mentor
- Personalized career roadmap
- Skill gap analysis
- AI career chat assistant

### 📊 Career Dashboard
- Resume score
- Interview progress
- Skill tracking
- Career insights


## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons

### Backend
- FastAPI
- SQLAlchemy
- SQLite
- AI APIs
